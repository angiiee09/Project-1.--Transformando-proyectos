# Project-1.--Transformando-proyectos
Este proyecto esta enfocado en brindar apoyo y soporte tanto a profesionales, autónomos y dueños de negocios que aun se encuentran un poco perdidos con las nuevas tecnologías y herramientas de IA actuales. Es normal sentirse un poco consternados al tomar decisiones sobre algo que aún no tenemos claro y que tenemos que tomarlas.
